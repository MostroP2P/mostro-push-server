Reqwest Client API (from Cargo.lock — reqwest 0.11.27):
- `reqwest::Client::builder()` returns `ClientBuilder`.
- `.connect_timeout(Duration)` — Set TCP connect timeout.
- `.timeout(Duration)` — Set total request timeout (including body reads).
- `.pool_idle_timeout(Option<Duration>)` — Set idle connection eviction timeout. Accepts `Some(Duration)` or `None`.
- `.build()` returns `Result<Client, reqwest::Error>`.
- `Client` is internally `Arc`-wrapped; `.clone()` is cheap. Wrapping in `Arc<Client>` is for type-system uniformity, not for performance.
